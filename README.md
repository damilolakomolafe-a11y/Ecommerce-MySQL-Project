# Ecommerce SQL Data Analysis Project 

## Project Overview
This project analyzes an Ecommerce database using SQL queries to generate business insights related to sales performance, customer behavior, employee performance, and product sales trends.
The project was completed as part of a SQL data analysis learning program.

## Database Used
**Database Name:** Ecommerce  

Tables Included:
- Customers  
- Orders  
- OrderDetails  
- Products  
- Employees  
- Categories  
- Suppliers  
- Shippers  

## Project Objectives
The goal of this project is to:

- Calculate total sales performance by employees  
- Identify top revenue generating customers  
- Analyze monthly and yearly sales trends  
- Measure order fulfilment performance  
- Analyze discount usage across products  
- Rank employees based on sales performance  
- Identify high value orders and product performance  

## SQL Skills Demonstrated
- Joins (INNER JOIN, LEFT JOIN)
- Aggregations (SUM, AVG, COUNT)
- Grouping and Filtering (GROUP BY, HAVING)
- Date Functions (YEAR, MONTH)
- Window Functions (RANK, PERCENT_RANK, LAG)
- Conditional Logic (CASE WHEN)
